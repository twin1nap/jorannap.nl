galge in python

In dit project moest ik galgje maken in python, dat betekent dus dat je het speelt in de CLI(commandline interface) / terminal, ik moest eerst pseudo-code schrijven en dan het programmeren in python, het spel had deze minimale eisen:

  • Invoer van het te raden woord.
  • Invoer van letters om het woord te raden.
  • Controle of de letters voorkomen in het woord
  • Een hint tonen uit hoeveel letters het woord besttat (streepjes).
  • De geraden letters van het woord onthullen.
  • Melding tonen waneer het woord geraden is.
  • Melding tonen waneer de speler veloren heeft.
  • Controle op invoer van het te raden woord.
  • Controle op invoer van elke geraden letter.

Het python bestand kan je vinden op github.
